# 과도한 동기화는 피하라
* 과도한 동기화는 성능을 떨어뜨리고, 교착상태(데드락)에 빠뜨리고, 심지어 에측할 수 없는 동작을 발생시킨다.
* 응답불가(데드락, 교착 상태)와 안전 실패(데이터 레이스)를 피하려면 동기화 메소드나 동기화 블록 안에서는 제어를 절대로 클라이언트에 양도하면 안 된다.
