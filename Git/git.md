## git 명령어들
### git init
* 수정내역이 저장되는 .git 디렉토리 생성
### git commit
* 각 버전의 변경사항을 기록
### git log
* commit 기록 조회
### git reset
* 과거 commit 기록으로 복원
### git branch
* 분기

## Git VS Github
* Git : 버전관리를 위한 소프트웨어
* Github : Git으로 저장돼서 원격전송된 내역들이 저장되는 공간을 제공하는 서비스
